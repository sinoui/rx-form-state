表单值处理

@sinoui/rx-form-state 的核心作用就是用来处理表单值的。本篇教程将会从以下方面一一介绍如何采用本库处理表单的值：

  • 表单域值绑定
  • 展现表单值
  • 表单域值关联
  • 全局值关联
  • 设置表单域的值

创建表单状态上下文

无论我们要收集表单值、表单验证还是提交表单，我们都需要先创建表单状态的上下文。通过useFormState可创建表单状态管理器。如下所示：

import { useFormState } from '@sinoui/rx-form-state';
function FormDemo() {
  const initialValue = {}; // 表单默认值
  const options = {}; // 创建表单状态管理器的配置
  const formState = useFormState(initialValue, options);
}

表单域值绑定

对于一个表单来说，它的值是对象结构的。如：

{
  "userName": "紫诺",
  "sex": "女"
}

如果我们要维护姓名和性别这两个字段，可以使用Field组件定义表单域。通过Field组件的name属性，可以将表单域组件和表单状态中的值进行绑定。如下所示，我们定义一个维护姓名字段值的表单域：

通过Field组件的name属性，可以将表单域组件和表单状态中的值进行绑定。

import React from 'react';
import { useFormState, Field, FormStateContext } from '@sinoui/rx-form-state';
function FormDemo() {
  const initialValues = {
    userName: '紫诺',
    sex: '女',
  };
  const formState = useFormState(initialValues);
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        姓名：
        <Field as="input" name="userName" />
        复制的姓名：
        <Field as="input" name="userName" />
      </form>
    </FormStateContext.Provider>
  );
}

效果如下：

姓名： 复制的姓名：

name属性可以指定任意深度的对象路径。例如，如果我们的表单值的结构如下：

{
  "userName": "紫诺",
  "address": {
    "city": "北京",
    "region": "海淀"
  }
}

那么关于地址的两个表单域的name可以如下表示：

<form>
  城市：
  <Field as="input" name="address.city" />
  区/县：
  <Field as="input" name="address.region" />
</form>

从 DOM 中提取值

Field 组件内置了从表单域组件值变化事件（ event ）中提取表单域值的默认规则：

  • 如果 event 是事件类型，则表单域值为 event.target.value；
  • 否则认为 event 是表单域的值。

如果默认规则不符合要求，可以查看自定义表单域中关于值提取的说明。

展现表单值

上一小节的示例中已经演示了采用FormValueMonitor组件来展现表单值。这一小节详细介绍用FormValueMonitor组件展现表单值。

FormValueMonitor组件会监控表单的值变化，一旦值发生变化，就会获取到新的表单值，并用新表单值作为参数调用children属性指定的渲染函数，展现出表单值。如下所示：

import React from 'react';
import {
  useFormState,
  Field,
  FormValueMonitor,
  FormStateContext,
} from '@sinoui/rx-form-state';
function FormDemo() {
  const initialValues = { userName: '紫诺', address: { city: '北京' } };
  const formState = useFormState(initialValues);
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        姓名：
        <Field as="input" name="userName" />
        所在城市：
        <Field as="input" name="address.city" />
        <FormValueMonitor>
          {(values) => (
            <div>
              填写的信息：{values.userName}@
              {values.address && values.address.city}
            </div>
          )}
        </FormValueMonitor>
      </form>
    </FormStateContext.Provider>
  );
}

效果如下：

姓名： 所在城市：
填写的信息：紫诺@ 北京

表单域值关联

上一节介绍的FormValueMonitor能展现表单的统计信息，但是如果需要将展现的表单统计信息也存储到表单值对象中，就可以采用表单域值关联。Field有两个属性：

  • relyFields - 定义表单域值关联计算逻辑依赖的其他表单域名称
  • relyFn - 定义表单域值关联计算的逻辑。它是一个接收整个表单值的函数，它的返回值将会作为该表单域的值。

例如：A = B + C这样的一个表单域值关联关系，首先我们需要在表单域 A 上定义relyFields为['B', 'C']，这样当B或者C表单域值发生变更时，就会调用A表单域的relyFn属性。

function parseNumber(str) {
  return str && !isNaN(parseInt(str)) ? parseInt(str) : 0;
}
function countA(values) {
  const { B, C } = values;
  return parseNumber(B) + parseNumber(C);
}
<Field name="A" as="input" type="number" relyFields={["B", "C"]} relyFn={countA} />
<Field name="B" as="input" type="number" />
<Field name="C" as="input" type="number" />

演示效果如下：

表单域值关联：A = B + C

A： B： C： 
() => {
  const formState = useFormState()
  function parseNumber(str) {
    return str && !isNaN(parseInt(str)) ? parseInt(str) : 0
  }
  function countA(values) {
    const { B, C } = values
    return parseNumber(B) + parseNumber(C)
  }
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        <h3>表单域值关联：A = B + C</h3>
        A：
        <Field
          name="A"
          as="input"
          type="number"
          relyFields={['B', 'C']}
          relyFn={countA}
        />
        B：
        <Field name="B" as="input" type="number" />
        C：
        <Field name="C" as="input" type="number" />
      </form>
    </FormStateContext.Provider>
  )
}

全局值关联

有时候，表单域值关联无法满足值关联的需求，比如A = B + C，而A不是一个可见的表单域元素。这时可以用一下全局值关联

import React from 'react';
import {
  useFormState,
  Field,
  FormValueMonitor,
  FormStateContext,
} from '@sinoui/rx-form-state';
// A = B + C
const abcRely = [
  'B',
  'C',
  (draft) => {
    if (draft.B && draft.C) {
      draft.A = draft.B + draft.C;
    }
  },
];
// E = D
const deRely = [
  'D',
  (draft) => {
    draft.E = draft.D;
  },
];
function FormDemo() {
  const initialValues = { userName: '紫诺' };
  const formState = useFormState(initialValues, {
    relys: [abcRely, deRely],
  });
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        B：
        <Field as="input" name="B" />
        C：
        <Field as="input" name="C" />
        D:
        <Field as="input" name="D" />
        <FormValueMonitor>
          {(values) => (
            <div>表单值：{JSON.stringify(values, undefined, 2)}</div>
          )}
        </FormValueMonitor>
      </form>
    </FormStateContext.Provider>
  );
}

效果如下：

B： C： D:
表单值：{}
() => {
  // A = B + C
  const abcRely = [
    'B',
    'C',
    draft => {
      if (draft.B && draft.C) {
        draft.A = draft.B + draft.C
      }
    },
  ]
  // E = D
  const deRely = [
    'D',
    draft => {
      draft.E = draft.D
    },
  ]
  const formState = useFormState(
    {},
    {
      relys: [abcRely, deRely],
    },
  )
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        B：
        <Field as="input" name="B" />
        C：
        <Field as="input" name="C" />
        D:
        <Field as="input" name="D" />
        <FormValueMonitor>
          {values => (
            <div>表单值：{JSON.stringify(values, undefined, 2)}</div>
          )}
        </FormValueMonitor>
      </form>
    </FormStateContext.Provider>
  )
}

设置表单域的值

当你需要在Field组件之外设置个别表单域的值，你可以调用formState.setFieldValue函数。如下所示：

import React from 'react';
import {
  useFormState,
  Field,
  FormValueMonitor,
  FormStateContext,
} from '@sinoui/rx-form-state';
function FormDemo() {
  const initialValues = { userName: '紫诺' };
  const formState = useFormState(initialValues);
  const handleClick = () => {
    formState.setFieldValue('userName', '张三');
  };
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        姓名：
        <Field as="input" name="userName" />
        <button onClick={handleClick}>将姓名变更为张三</button>
      </form>
    </FormStateContext.Provider>
  );
}

设置整个表单值

表单的初始值有可能需要通过 Ajax 从服务器端加载到。例如：

import React from 'react';
import {
  useFormState,
  Field,
  FormValueMonitor,
  FormStateContext,
} from '@sinoui/web-forms';
import useRestItemApi from '@sinoui/use-rest-item-api';
function FormDemo() {
  const dataSource = useRestItemApi('/users', '1');
  const formState = useFormState(dataSource.data);
  const handleClick = () => {
    formState.setFieldValue('userName', '张三');
  };
  return (
    <FormStateContext.Provider value={formState}>
      <form>
        姓名：
        <Field as="input" name="userName" />
        <button onClick={handleClick}>将姓名变更为张三</button>
      </form>
    </FormStateContext.Provider>
  );
}
